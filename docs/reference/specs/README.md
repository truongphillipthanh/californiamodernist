# Design Specifications

This directory contains wireframes, mockups, and design specifications for the CMD System.

## File Organization

- Place wireframes as PNG or PDF files
- Name files descriptively: `{feature}-{view}-wireframe.png`
- Include version numbers for iterations: `{feature}-v2.png`

## Reference

Always reference CMD_STYLE_GUIDE.md in the project root for:
- Typography specifications
- Color palette
- Spacing system
- Component patterns
