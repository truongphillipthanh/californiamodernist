# CMD System Documentation

## Structure

- `/reference` — Visual references for UI development
  - `/ui-states` — Screenshots of current and target UI states
  - `/specs` — Wireframes, mockups, design specifications
- `/architecture` — System architecture documentation

## For Claude Instances

When implementing UI changes, reference images in `/reference/ui-states/`
to understand the target state. Each image has annotations describing
required changes.
