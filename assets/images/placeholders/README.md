# Placeholder Images

Placeholder images for development and testing.

## Naming Convention

Use format: `placeholder-{width}x{height}.jpg`

Example: `placeholder-1600x900.jpg`

## Specifications

- Common aspect ratios: 16:9, 4:3, 1:1
- Grey background (#CCCCCC)
- CMD watermark centered
- Dimensions visible in corner
- Optimized file size
