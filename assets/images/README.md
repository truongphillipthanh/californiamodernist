# CMD Images

Photography and imagery for the CMD System.

## Directories

- `/projects` — Project photography organized by project name
- `/team` — Team member headshots and profile photos
- `/placeholders` — Placeholder images for development

## Image Optimization

- Compress all images for web
- Use JPG for photos (quality 80-85%)
- Use PNG for graphics with transparency
- Maximum width: 2000px for full-screen images
- Provide @2x versions for retina displays
