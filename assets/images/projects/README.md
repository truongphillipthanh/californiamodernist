# Project Images

Each subfolder contains images for a specific project.

## Naming Convention
```
{project}/
├── hero.jpg           # Primary/featured image
├── exterior-01.jpg    # Exterior shots
├── exterior-02.jpg
├── interior-01.jpg    # Interior shots
├── render-01.jpg      # 3D renders
├── elevation-01.jpg   # 2D elevations
└── README.md          # Project-specific notes
```
