# CMD Logo Files

Official logo files for California Modernist Design.

## File Formats

- `cmd-logo.svg` — Primary logo (scalable vector)
- `cmd-logo@2x.png` — High-resolution raster fallback
- `cmd-logo-white.svg` — White version for dark backgrounds
- `cmd-icon.svg` — Icon-only variant

## Usage

- Use SVG files for web when possible
- Maintain minimum size of 120px wide
- Preserve aspect ratio
- Allow clear space equal to the height of "CMD" letters
