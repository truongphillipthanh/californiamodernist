# CMD System Assets

## Structure

- `/branding` — Official CMD branding materials
  - `/logo` — Logo files (SVG preferred, PNG for fallback)
  - `/colors` — Color palette swatches
- `/images` — Photography and imagery
  - `/projects` — Project photos organized by project name
  - `/team` — Team member headshots
  - `/placeholders` — Development placeholder images

## Image Guidelines

- Photos: JPG, max 2000px wide, optimized for web
- Logos: SVG preferred, PNG@2x for raster needs
- Placeholders: 16:9 aspect ratio, grey background, CMD watermark
